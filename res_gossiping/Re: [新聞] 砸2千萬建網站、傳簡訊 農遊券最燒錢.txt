: 三倍券推出後，文化部、體育署、農委會、客委會也紛紛推出各種加碼券，其中又以農遊
: 券的「成本」最高，網站建置再加上行銷、中籤簡訊費用，合計高達二千萬餘元，動滋券
: 也花了一一○○萬元建置網站；至於藝FUN券的App建置系統花費七四五萬元，客委會「最
: 省」，利用原有LINE平台，只動用一五五萬元的環境系統設計費。
: 農委會輔導處休閒產業科長楊欣佳說，早在二月初就規畫好農遊券的完整架構，五月公開
: 招標，整個標案金額約一五二○萬元，費用內容包括登記、行銷、業者說明會、客服等；
: 發放中籤簡訊費用採實支實付，先由廠商代墊，估有四百多萬元。
: 不過農遊券發放規模為各部會最大，卻只花不到半年時間規畫，是否太過倉促？楊欣佳說
: ，農委會首次發放電子票券，盡量謹慎思考，但確實發生很多沒有預期到的狀況，若能夠
: 調整的部分就會改善。

: 至於中籤率超低的客庄券，客委會產業經濟處科長楊秀禎指出，從三月就開始著手籌辦、
: 招標承接浪漫客庄旅遊券的廠商，接著五月廿七日決標，到七月十五日發放，總共歷時三
: 個半月的籌備期，考量民眾使用方便性，因此採用既有的LINE官方群組登記、發放，環境
: 系統設計費花了一五五萬元，另又編列約四百萬元在招募人力、報名窗口建置、商家教育
: 訓練等。

: 文化部表示，政府二月提出紓困振興方案時就已開始規畫藝FUN券，管理平台建置費用共
: 七四五萬元，其中包含店家與民眾兩端App、管理系統平台、藝文店家地圖製作、數據分
: 析功能等。
: 原訂在七月卅日開放領券的動滋券，確定延期到八月六日發放。體育署副署長王水文表示
: ，將依照協議書及計畫書所列工作項目、時程完成各項工作，若有未能於規定時程辦理，
: 將減價或罰款；據了解動滋券在網站建置費用上花費為一一○○萬元。
